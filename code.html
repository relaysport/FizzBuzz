// fizzFizz function

function fizzFizz(value1, value2) {
    let returnValue = [];
    for (let i = 1; i <= 100; i++) {
        returnValue.push((i % value1 == 0 ? "Fizz" : "") + (i % value2 == 0 ? "Buzz" : "") || i);
    }

    return returnValue;

}

// buzzBuzz function

function buzzBuzz() {
    let value1 = document.getElementById("fizzValue").value;
    let value2 = document.getElementById("buzzValue").value;

    let templateHTML = document.getElementById("template-row-items")
    let resultsHTML = "";
    let output = fizzFizz(value1, value2);

    for (i = 0; i < output.length; i += 5) {
        resultsHTML += templateHTML.innerHTML.replace('{{val1}}', output[i])
            .replace('{{val2}}', output[i + 1])
            .replace('{{val3}}', output[i + 2])
            .replace('{{val4}}', output[i + 3])
            .replace('{{val5}}', output[i + 4]);
    }

    document.getElementById("results").innerHTML = resultsHTML;
}

fizzFizz

The fizzFizz function pulls the user inputs from the function buzzBuzz and orders them as value1 and value2. An incrementing for loop runs from 1 to 100 to display 'Fizz' when the given number, i, is divisible by value1. The same argument is used for value2 and 'Buzz'. If both conditions are met, the output is 'FizzBuzz'. If neither of the conditions are met, the output is the given number.

buzzBuzz

The primary use of the second function is to display the game in a table. The first few lines are included for necessity; 10 and 11 pass the user inputs to fizzFizz, lines 13 through 15 declare variables and information from fizzFizz needed for the following for loop to properly run. The incrementing for loop runs for the entirety of the fizzFizz data (from 1 to 100), but only gives five values per loop. Running for five values allows the for loop to align with the HTML template, displaying five outputs for each row in the table. The template placeholders are replaced by fizzFizz outputs to create a table!
